GetMultiGroupPropertyRelations()

GetMultiGroupPropertyRelations(array  $gids) : array

Get a sorted list of properties for a list of gids.

This method will find all of the properties associated with all of the specified groups and then sort them by their name, and adjust priorities to the highest level.

Keeping in mind that users can belong to multiple groups, and because different properties can be associated with different groups or given different priorities, this method resolves all of that into a sorted list of relations

Parameters

array $gids

Returns

array —

of group property relation records (hashes). giving the property name, sorting and readonly statuses
